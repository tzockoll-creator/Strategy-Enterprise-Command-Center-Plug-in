# Weekly Business Ops Review

Execute the Enterprise Command Center skill to run a comprehensive weekly business operations review.

## Execution Instructions

1. **Verify MCP connections** - Ensure Strategy MCP, Asana, Atlassian, HubSpot, Slack, Canva, and Figma are connected
2. **Run Round 1: Sales Intelligence** - Query Strategy for pipeline, at-risk deals, revenue signals
3. **Route to HubSpot/Asana** - Update deals, create follow-up tasks
4. **Run Round 2: Operations Intelligence** - Query Strategy for blockers, SLA status, capacity
5. **Route to Jira/Confluence/Asana** - Create issues, documentation, tasks
6. **Run Round 3: Executive Intelligence** - Analyze dashboards for visual patterns
7. **Route to Slack/Canva/Figma** - Post summary, generate presentation, update designs
